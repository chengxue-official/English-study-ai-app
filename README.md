# 英语应试 App 实验项目

这是一个基于 React + Vite (Client) 和 Node.js (Server) 开发的英语学习应用。

## 主要功能
- 单词查询（支持本地离线词典、在线 API 与 AI 语境分析）
- 点击查词（支持释义与词形变化中的单词点击查询）
- 单词复习系统（艾宾浩斯记忆曲线）
- Android 客户端支持 (Capacitor)

## 仓库说明
- **Gitee (Origin)**: 主仓库，国内访问优化。
- **GitHub**: 备份仓库。

## 鸣谢
本项目使用了以下优秀的开源项目：
- **[ECDICT](https://github.com/skywind3000/ECDICT)**: 简明英汉字典数据库，为本项目提供强大的离线词库支持。
- **[PaddleOCR](https://github.com/PaddlePaddle/PaddleOCR)**: 优秀的 OCR 识别库，本项目通过其 API 实现图片文字识别功能。
