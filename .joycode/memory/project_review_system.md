## 复习系统改造需求

### 1. 复习卡片展示升级
- 显示：单词 + 音标 + 发音按钮（在线真人发音）
- 用户点击发音按钮播放真人发音

### 2. 不认识按钮行为修改
- 当前：直接加入队列末尾
- 新：展示该单词的完整释义面板（词性+意思），自动播放一遍真人发音
- 用户看完释义后点击"下一个"继续
- 该单词本次复习仍不再出现（标记为不认识但移出本轮）

### 3. 认识按钮行为
- 直接消掉这个词，本轮不再出现
- 标记复习成功

### 4. 10个一组复习
- 每轮最多10个单词一组
- 一组完成后显示"拼写测试"选项
- 拼写测试：给出中文释义，要求用户输入英文单词

### 5. 艾宾浩斯记忆曲线
- 每个收藏单词增加下次复习时间字段
- 基于复习次数计算下次复习时间间隔
- 1次: 1天后, 2次: 2天后, 3次: 4天后, 4次: 7天后, 5次: 15天后, 6次+: 30天后
- 复习界面优先展示到期需要复习的单词
