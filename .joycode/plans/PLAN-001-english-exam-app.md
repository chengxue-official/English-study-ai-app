# 高中英语应试App - 项目规划

## 任务概述
开发一款面向高中生的英语应试辅助App，核心功能包括：文章导入与自动翻译、单词点击查询、长难句拆解分析、短语/搭配/语法标注。

**目标形态**：最终打包为独立exe(桌面端) + apk(移动端)，目前先以Web形式开发，功能稳定后再决定打包方案(Tauri/Electron+Capacitor)。

**移动端适配**：已实现响应式布局，手机端可通过局域网访问调试。

**更新策略**：大版本(原生层变更)发布新exe/apk；小功能更新通过应用内热更新机制，从远程拉取最新前端资源包自动替换，无需重新安装。封装时再实现。

## 技术栈选型

### 前端
- **框架**: React + Vite (现代SPA，热更新，开发体验好)
- **UI库**: Ant Design / Tailwind CSS (快速搭建界面)
- **状态管理**: Zustand (轻量级)

### 后端
- **框架**: Node.js + Express / Fastify
- **NLP/AI服务**:
  - 翻译: 接入大模型API (支持OpenAI兼容接口)
  - 分词/词性标注: compromise.js / natural (前端NLP)
  - 词典数据: 本地SQLite + 在线词典API备用

### 数据库
- **SQLite**: 本地词典库、用户数据
- **IndexedDB**: 前端缓存(文章、查询历史)

## 关键技术挑战
1. **长难句识别**: 需要NLP分词+句法分析，可先用规则+AI混合方案
2. **短语识别**: 需要构建高中英语常用短语库
3. **语法标注**: 需要语法规则引擎或AI辅助
4. **翻译质量**: 大模型API翻译质量较好，但需控制成本
